And here is how to build it:

1) empty lines are ignored, but put spaces between sections to help me debug.

2) for each line
Note that this is also an EDI type forman. each line begins with a key that tells what it is and what follows, fields separated with *.

So, there should not be '*' in the data. Period simple. I do not know how to solve that bad decision of  me, but for now make sure to filter out any * you see in data.

So instructions to build the file:

BRICKCONFIG*2  <--- hard coded begin marker


ASSETS  <----  section start
AS*121*TRUCK*123  <--------- [AS]asset record * asset_id (not theuuid) * layout_name * asset_tag  

AS*331*MIXER*123
AS*332*MIXER*123456
AS*333*MIXER*123456
